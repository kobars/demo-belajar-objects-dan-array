## Belajar mutability, immutability, filter, map, reduce

#### Untuk menjalankan project:

Menggunakan npm

```bash
npm install
npm run dev
```

atau menggunakan yarn

```bash
yarn install
yarn run dev
```

lalu buka http://localhost:3000/

#### Product objects

Buka folder utils/product.js

#### Map, filter, reduce

Buka folder utils/index.js
